# TENSAI - Quiz Battle Game

## Overview
This is a browser-based quiz battle game created as a group project.

Players choose a subject (Japanese, Math, Science, Social Studies) and challenge a boss through quiz questions.

## Tech Stack
- HTML
- CSS
- JavaScript (DOM manipulation / event handling)

## My Role

- Debugging

## Features
- Subject selection
- Quiz system
- Answer validation
- Boss battle format
